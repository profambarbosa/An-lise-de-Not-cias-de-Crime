# An-lise-de-Not-cias-de-Crime
Criação de um prompt que seja hábil a promover uma análise mais profícua possível de notícias de crimes para se estabelecer uma hipótese criminal viável para solução do caso apresentado com indicação de autoria, materialidade delitiva e circunstâncias relativas ao tempo, lugar, modo, meios que retratam a prática do crime sob investigação.
