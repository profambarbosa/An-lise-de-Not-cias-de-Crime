# An-lise-de-Not-cias-de-Crime
Criação de um prompt que seja hábil a promover uma análise mais profícua possível de notícias de crimes para se estabelecer uma hipótese criminal viável para solução do caso apresentado com estabelecimento de autoria, materialidade  delitiva e circunstâncias que retratam a prática do crime sob investigação.
